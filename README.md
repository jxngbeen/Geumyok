# 금욕

금연처럼 특정 습관을 끊거나 참는 도전을 시작하고, 매일 성공 또는 실패를 기록하는 iOS 앱입니다.

## 현재 구현된 첫 버전

- SwiftUI 기반 iOS 앱 프로젝트
- 목표 일수 선택 후 도전 시작
- 중앙 원형 진행률 안에 성공 일수 표시
- 오늘 성공 / 실패 기록 버튼
- 실패 또는 목표 달성 시 결과 화면 표시
- `UserDefaults` 기반 로컬 저장

## 열기

Xcode에서 `Geumyok.xcodeproj`를 열고 iPhone 시뮬레이터로 실행하면 됩니다.

## GitHub 연동

현재 폴더는 Git 저장소로 초기화되어 있습니다. GitHub 저장소를 만든 뒤 원격 주소를 연결하면 됩니다.

```sh
git remote add origin <GitHub repository URL>
git push -u origin main
```
