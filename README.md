# 금욕

금욕은 금연, 야식 끊기, SNS 줄이기처럼 계속 이어가고 싶은 목표를 만들고 매일 성공 여부를 기록하는 SwiftUI 기반 iOS 앱입니다. 여러 개의 금욕을 동시에 관리하고, 진행 중인 금욕과 종료된 금욕을 나눠 볼 수 있습니다.

## 주요 기능

- 금욕 생성 시 목표명과 목표 일수를 설정
- Apple 기본 휠과 숫자 입력으로 목표 일수 선택
- 여러 개의 금욕을 진행 중 / 지난 금욕으로 분리 관리
- 홈 화면에는 각 목록을 최근 5개까지만 표시하고, 더보기 화면에서 전체 확인
- 진행률 원형 링으로 성공 일수와 목표 달성률 표시
- 오늘 성공 기록 및 성공 기록 취소
- 오늘 성공 후 실패 기록 버튼 비활성화
- 실패 또는 목표 달성 시 종료된 금욕으로 이동
- 시작일, 종료 예정일, 실제 종료일 표시
- 금욕 상세 화면에서 목표명 바로 수정
- 실패한 금욕은 같은 조건으로 다시 시작할 수 있도록 생성창에 미리 채우기
- 같은 이름으로 다시 만들면 `2회차`, `3회차`처럼 자동 회차 이름 적용
- 목록에서 슬라이드 삭제
- 더보기 화면에서 여러 금욕을 선택하거나 쓸어서 선택한 뒤 한 번에 삭제
- `UserDefaults` 기반 로컬 저장

## 화면 구성

- 홈: 진행 중 금욕과 지난 금욕 목록 확인, 새 금욕 생성
- 금욕 상세: 진행률, 성공 일수, 남은 날, 날짜 정보, 성공/실패 기록
- 더보기: 진행 중 금욕 또는 지난 금욕 전체 목록 확인 및 다중 삭제
- 생성 화면: 목표명과 목표 일수 설정

## 기술 스택

- Swift
- SwiftUI
- UIKit 연동: 키보드 완료 버튼, 기본 휠 피커
- UserDefaults 로컬 저장

## 실행 방법

1. Xcode에서 `Geumyok.xcodeproj`를 엽니다.
2. iPhone 시뮬레이터를 선택합니다.
3. Run 버튼을 눌러 앱을 실행합니다.
