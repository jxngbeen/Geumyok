# 금욕 앱 아이콘 레이어 PNG

기존 앱 아이콘을 Apple Icon Composer에서 다시 조합하기 쉽도록 PNG 레이어로 분리한 폴더입니다.

권장 배치 순서:

1. `01-background.png`
2. `02-mint-accent.png`
3. `03-white-symbol.png`

보조 파일:

- `00-original-reference.png`: 기존 아이콘 원본 확인용
- `04-foreground-combined.png`: 민트/흰색 전경을 한 번에 올려보고 싶을 때 사용

주의:

- 원본 PNG에서 색상 기준으로 분리한 파일이라 완전한 원본 벡터 레이어는 아닙니다.
- Icon Composer에서 세부 깊이감이나 glass 효과를 조정하면서 쓰는 출발점으로 보면 됩니다.
