# 금욕 새 앱 아이콘 레이어

Icon Composer에서 새로 조합하기 좋도록 깨끗하게 다시 그린 PNG 레이어입니다. 기존 아이콘 PNG를 억지로 분리한 파일이 아니라, 같은 컨셉을 새 도형으로 재구성했습니다.

권장 배치 순서:

1. `01-background.png`
2. `02-mint-glow.png`
3. `00-optional-soft-shadow.png` (선택)
4. `03-progress-ring.png`
5. `04-white-flame.png`

확인용:

- `preview-composite.png`: 위 레이어를 합친 미리보기

메모:

- 모든 PNG는 1024x1024 RGBA입니다.
- Icon Composer에서 glass, depth, shadow 값은 다시 조절해도 됩니다.
- 프로젝트의 실제 앱 아이콘 asset은 건드리지 않았습니다.
