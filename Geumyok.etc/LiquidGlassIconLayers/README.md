# 리퀴드 글래스(Liquid Glass) 스타일 앱 아이콘 레이어

Icon Compressor 등에서 조합하기 위해 새로 제작된 "리퀴드 글래스" 테마의 PNG 레이어들입니다.
금욕(Geumyok) 앱의 시그니처인 '불꽃' 모티프를 투명하고 고급스러운 글래스 질감으로 재해석했습니다.

## 층별 안내 (아래에서 위 순서)

1. `01-background.png`: 부드럽고 묵직한 어두운 액체 금속/글래스 배경
2. `02-mint-glow.png`: 글래스 불꽃을 뒤에서 은은하게 받쳐주는 민트색 오라(Aura)
3. `03-progress-ring.png`: 앱의 진행도(Progress Ring)를 상징하는 원형 글래스 링
4. `04-glass-flame.png`: 앱의 핵심 심볼인 입체적인 리퀴드 글래스 불꽃 아이콘

## 확인용 미리보기
- `preview-composite.png`: 위 레이어들을 조합했을 때의 최종 렌더링 예상도

## 활용 방법
- 각 PNG 파일은 배경이 제거된(투명한 알파 채널) 진짜 레이어입니다.
- Icon Composer 앱이나 Figma, Photoshop 등에 순서대로 드래그 앤 드롭하여 Parallax 효과나 Depth를 조절할 수 있습니다.
- 01번(배경) 레이어 위에 02, 03, 04번 레이어를 차례대로 올리면 검은 배경이 겹치지 않고 온전히 글래스와 빛이 합성됩니다.
